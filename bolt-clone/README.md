# bolt-clone

Recreation of the Bolt ride-hailing flow: map home screen with a draggable sheet, ride options with payment methods, ride history, account.

Expo 57, expo-router, react-native-maps, @gorhom/bottom-sheet, Reanimated, NativeWind.

## Run

```
npm install
npx expo start
```

Needs a device/simulator with maps support.
