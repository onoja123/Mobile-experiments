# trend-catcher

Streetwear discovery app concept. An explore feed with big lowercase editorial headlines over horizontal carousels, and a collections screen where the titles slide over the images with a parallax effect as you scroll.

Expo 57, Reanimated, SVG, NativeWind.

## Run

```
npm install
npx expo start
```

Card images load from Unsplash, so it needs a network connection.
