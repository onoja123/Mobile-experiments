# Mobile experiments

This is where I keep my React Native / Expo experiments — animations, gestures, and UI ideas I try out and share. Each folder is a standalone Expo app with its own README.

| Project | What it is |
|---|---|
| [ai-chat-ui](ai-chat-ui) | Chat UI with a drawer sidebar and a send-button morph animation |
| [audiobook-player](audiobook-player) | Mini player that expands into a full player sheet |
| [bolt-clone](bolt-clone) | Bolt ride-hailing flow — map, ride options, payments |
| [bookmark-dissolve](bookmark-dissolve) | Cards that dissolve into particles on delete (Skia) |
| [crypto-wallet](crypto-wallet) | Balance digits that scramble and lock in on pull-to-refresh |
| [destination-card-expand](destination-card-expand) | Travel cards that expand into a detail sheet |
| [fold-to-delete](fold-to-delete) | Drag a folder to the delete zone and it folds away |
| [gooey-fab](gooey-fab) | FAB that splits into actions with a liquid metaball effect |
| [hotel-booking](hotel-booking) | Hotel search with an orbiting photo onboarding |
| [learning-chat](learning-chat) | Topic cards and a scripted practice chat |
| [neobank-wallet](neobank-wallet) | Card stack that expands with a spring animation |
| [retro-terminal](retro-terminal) | Terminal / sci-fi styled animated screens |
| [spotify-onboarding](spotify-onboarding) | Connect flow with an arcing card marquee |
| [sticky-note-peel](sticky-note-peel) | Notes that peel and fold under your finger |
| [wallet-card-gestures](wallet-card-gestures) | Gesture-driven credit card stack |
| [workflow-canvas](workflow-canvas) | Node canvas UI for a workflow automation tool |

Every project runs the same way:

```
npm install
npx expo start
```

I post these on X as I build them — [x.com/iam_the_code](https://x.com/iam_the_code). More at [iamthecode.xyz](https://iamthecode.xyz).
